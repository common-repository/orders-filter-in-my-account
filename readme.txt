== Description ==

Add a complete filter section in top of orders page in woocommerce my account automatically.

== Installation ==

1. Upload `Orders filter in my account` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Use of plugin ==

Activate it and enjoy.
Also you can hide/show filter element in plugin manage page (settings submenu)

== Frequently Asked Questions  ==

= Can I hide or show the desired fields?  =

 Yes, Just go to the plugin settings panel and select any field you want to display.
= Will this plugin slow down my website speed?  =

 No, the plugin don't slow your website. plugin's assets just loaded in orders page and they are very small and optimized.
